Шпаргалки

без них никуда =)
  • HTML Book - популярный справочник по HTML/CSS
  • Bootstrap 4 - таблица классов библиотеки Twitter Bootstrap
  • Emmet - ускоритель написания кода присутствующий во всех популярных редакторах
  • VSСode shortcuts - комбинации горячих клавиш для VSCode (Linux)
  • Flexbox - отличный гайд по использованию flex элементов
  • Boilerplate - универсальная заготовка для сайта, содержит необходимую структуру файлов и папок и актуальный набор исправлений для браузеров
  • WP-kama - все что вы хотели узнать о WordPress, но не знали где искать
  • RegEx - регулярные выражение в примерах, то что нам всем нужно знать

Инструментарий

все работает из браузера
  • Avocode - позволяет разобрать PSD макет на стили и блоки с размерами, две недели бесплатного пользования
  • PSDetch - бесплатный аналог предыдущего инструмента (чуть менее красивый)
  • Colorscheme - генератор цветовой палитры сайта, простенько, но со вкусом
  • Modernizer - инструмент позволяющий подстроить сайт под возможности браузера пользователя
  • Can I Use - проверка распространенности элемента, стиля или атрибута в различных браузерах
  • Code Generator - генератор различных элементов стилей по заданным параметрам, можно использовать для обучения или просто копипастить
  • Flexplorer - визуализатор возможностей flex
  • Masonry - сетка для многоколоночного расположения элементов страницы
  • Grid Template - создает 8, 9 и 12 колоночный PSD/PNG/CSS шаблон для сайта
  • Codyhouse - фреймворк с подоборкой наиболее распространенных элементов для создания сайта
  • Grid Generator - онлайн конструктор сетки
  • Transfonter - конвертирует загружаемый шрифт в нужный формат
  • CSSmatic - визуальный редактор для CSS
  • Animate CSS - библиотека различных анимаций
  • PNG to SVG - конвертер изображений (удобно для создания иконок)

Игры

не слишком интересные, зато поучительные
  • Flexbox Defense - постройте линию обороны с помощью flex
  • Grid Garden - вырастите сад с помощью grid
  • Flexbox Froggy - заселите пруд (хотя бы лягушатами)
  • CSS Diner - и накройте стол с помощью селекторов
